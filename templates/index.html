This demo enables you to quickly parse large text. By providing the large text input and your OpenAI API key the content is being analyzed by ChatGPT and you will receive a summary and an explanation of the entire content of the parsing large amount of text.
Note, that the demo is programmed, so that it compiles all information from the content even if exceeds the OpenAI API token limit which is currently at 4096 characters. For more information have a look at the demo's GitHub page: https://github.com/edgewaysai/TextAnalyzer
Depending on the size of the content it will take a while to generate the output

You can expand the text input field by dragging the lower right corner of the field
